# Genetic Lottery

A Timberborn mod that gives each beaver a random lifespan modifier at birth.

## Features

- Every beaver created receives a random Life Expectancy bonus between -10% and +10%.
- Some beavers live longer, some shorter â€” adding natural variation to your colony.
- The bonus is applied through the game's existing bonus system and affects the beaver for its entire life.

## Installation

Copy the mod folder to your Timberborn mods directory:
- **macOS**: `~/Documents/Timberborn/Mods/GeneticLottery/`
- **Windows**: `%USERPROFILE%\Documents\Timberborn\Mods\GeneticLottery\`
- **Linux**: `~/.config/unity3d/Mechanistry/Timberborn/Mods/GeneticLottery/`

## Compatibility

- Timberborn v1.0.0.0+
- Works with all factions
