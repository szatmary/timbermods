# Dynamite Rubble

A Timberborn mod that makes dynamite and tunnel explosions leave behind Dirt rubble.

## Features

- **Dynamite**: Produces 3 Dirt per terrain block actually destroyed. A triple dynamite on solid ground drops 9 Dirt, but on a 1-block overhang it only drops 3.
- **Tunnels**: Produces 3 Dirt when a tunnel finishes construction.

## Installation

Copy the mod folder to your Timberborn mods directory:
- **macOS**: `~/Documents/Timberborn/Mods/DynamiteRubble/`
- **Windows**: `%USERPROFILE%\Documents\Timberborn\Mods\DynamiteRubble\`
- **Linux**: `~/.config/unity3d/Mechanistry/Timberborn/Mods/DynamiteRubble/`

## Compatibility

- Timberborn v1.0.0.0+
- Works with all factions
